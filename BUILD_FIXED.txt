ğŸ“š WHAT WAS THE PROBLEM?

Razor doesn't allow complex C# expressions directly in Style attributes.

Wrong:  Style="color: @GetTopicColor(...);"
Right:  Style="@($"color: {GetTopicColor(...)};")"

The fix wraps the entire style string in a C# interpolated string 
using @($"...{expression}...") syntax.
