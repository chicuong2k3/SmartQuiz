
# Target segment: 

- Small rice restaurants
- Small eateries

# Key Features

- As a cashier, I want to create an order with a few taps so that I can serve customers quickly.
- As a cashier, I want a quick sale mode so that I don't need to enter unnecessary information.
- As a cashier, I want large buttons and fonts so that the system is easy to use during busy hours.

- As a restaurant owner, I want to configure a daily menu so that I can change dishes every day.
- As a restaurant owner, I want to copy yesterday's menu so that I don't need to recreate it.
- As a cashier, I want out-of-stock items hidden so that I don't sell unavailable dishes.
- As a cashier, I want to view today's orders so that I can track what has been sold.

- As staff, I want to add notes to orders so that special requests are handled correctly.

- As kitchen staff, I want to see new orders in real time so that food preparation starts immediately.
- As kitchen staff, I want to update order status so that the front counter knows the progress.

- As an owner, I want to create user accounts so that staff access is controlled.
- As the system, I want to restrict features by role so that users only see what they need.

- As a customer, I want to scan a QR code to view the menu so that ordering is easier.
- As a cashier, I want QR orders to appear on POS so that they can be processed normally.

- As a restaurant owner, I want to close a shift and see cash differences so that I can detect mistakes.

- As a restaurant owner, I want to see daily revenue so that I know how the business performed.
- As a restaurant owner, I want to see top-selling items so that I know which dishes perform best.

- As a cashier, I want to continue selling when the internet is down so that business is not interrupted.
- As the system, I want to sync offline orders automatically so that data is consistent.
- As a restaurant owner, I want to install the system like an app so that it feels native.
